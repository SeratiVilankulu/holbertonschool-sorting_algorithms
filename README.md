0 - Bubble sort

3 - Quick sort
4 - Shell sort - Knuth Sequence

Advanced Tasks:

5 - Cocktail shaker sort
6 - Counting sort
7 - Merge sort

9 - Radix sort
10 - Bitonic sort
11 - Quick Sort - Hoare Partition scheme
12 - Dealer
